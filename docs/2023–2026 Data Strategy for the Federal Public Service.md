## Framework

## Guiding Principles
1. Purposeful
2. Client-centred
3. Trusted
4. Ethical
5. Open
6. Enabling
## Desired Outcomes
1. Effective, equitable, ethical and inclusive services, programs and policy
2. Greater public value from data
3. Enhanced evidence-informed decision-making
4. Support for Indigenous data sovereignty
